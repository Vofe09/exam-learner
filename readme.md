# Exam Learner

Веб-сайт построенный для студентов и школьников, помогающий им в зубрежке ответов для экзаменов. 

## Возможности

- загрузка тестов из JSON
- случайный порядок вопросов
- выбор ответа из нескольких вариантов
- счёт, прогресс и число показанных вопросов

## Как использовать

1. Откройте `index.html` в браузере.
2. Добавьте тесты в папку `tests/`.
3. Укажите список тестов в `tests/index.json`.
4. Выберите тест и нажмите **«Загрузить тест»**.

## Формат теста

```json
{
  "title": "Математика",
  "questions": [
    {
      "text": "Сколько будет 2 + 2?",
      "options": ["3", "4", "5", "22"],
      "correctIndex": 1
    }
  ]
}
```

## Структура

```text
index.html
tests/
  index.json
  math.json
  history.json
```
